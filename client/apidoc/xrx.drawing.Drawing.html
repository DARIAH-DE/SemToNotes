Class: Drawing

.drawing. Drawing

new Drawing(element, opt_engine, opt_force)

A class representing a drawing canvas. The drawing canvas can have a background image and thereby can serve as an image annotation tool.
Parameters:
Name Type Description
element DOMElement The HTML element used to install the canvas.
opt_engine string The name of the rendering engine.
opt_force boolean Whether to force Raphael rendering. For debugging only.
