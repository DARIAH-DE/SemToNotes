Browser Support

  • Shapes can be rendered with three different browser engines: Canvas, SVG and VML.
  • If no special rendering engine is needed, the drawing canvas itself searches for the best engine available, Canvas, SVG, or VML, in this order.
  • In seldom cases it might be useful to force a drawing canvas to use a specific rendering engine. Rendering engines can be forced within the construction of a drawing canvas object, e.g., by calling new xrx.drawing.Drawing(element1, xrx.engine.CANVAS), new xrx.drawing.Drawing(element2, xrx.engine.SVG) or new xrx.drawing.Drawing(element3, xrx.engine.VML).
  • It is recommended to wrap a constructor call with a xrx.canvas.isSupported(), xrx.svg.isSupported() or xrx.vml.isSupported() condition if one forces a special rendering engine.
