Selecting Shapes

  • In order to style shapes differently when selected with touch or mouse, turn a drawing canvas over into mode select, e.g., with drawing1.setModeSelect().
  • Each shape object automatically has a helper shape attached called selectable that can be reached with the getSelectable() function. One can define the style of a selected shape, e.g., with rect.getSelectable().setStrokeColor('#99CCFF') or circle.getSelectable().setFillColor('#99CCFF').
  • Besides selecting a shape with touch or mouse, one can also programmatically turn a shape into selected state. The left example sets the first shape into selected mode by calling drawing1.setSelected(drawing1.getShapes()[0]).
  • The right example demonstrates that one can remove a shape from the drawing canvas by selecting it with touch or mouse and by implementing a button. Removing a shape is, failing that, realized with the removeShape() function inside the button's click handler: <button onclick="window.drawing.removeShape(window.drawing.getSelectedShape());">Remove selected shape</button>.
