Initialization

  • The size of a drawing canvas can be defined with CSS rules or programmatically.
  • CSS sizes can be absolute values such as #drawing1 { width: 200px; height: 200px; } or relative values, e.g., #drawing2 { width: 50%; min-height: 150px; }. In the case of relative values, the drawing canvas automatically resizes the internal rendering engine element (<canvas>, <svg>, <vml:group>) whenever the size of the browser window changes.
  • One can set the size programmatically with a drawing3.setSize(300, 200) function that expects the width and height in pixel. Programmatic size definitions overwrite CSS rules. Again, the drawing canvas automatically resizes the internal rendering engine element..
  • One can define all kinds of CSS styling rules for a drawing canvas such as #drawing4 { background-color: #D1F0FF; margin: 20px; padding: 20px; }.
  • It is not recommended to apply any CSS rules to the rendering engine elements. Use the drawing canvas element for styling instead.
